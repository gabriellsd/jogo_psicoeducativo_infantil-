# O Cérebro Preocupado

Jogo psicoeducativo infantil sobre ansiedade e o “Cérebro Protetor”.

## Desenvolvimento local

```bash
npm install
npm run dev
```

## Deploy (Vercel)

O projeto usa Vite + React. A Vercel detecta o framework automaticamente; o arquivo `vercel.json` garante o roteamento da SPA.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/gabriellsd/jogo_psicoeducativo_infantil-)
